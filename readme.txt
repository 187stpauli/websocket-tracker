# Мониторинг пулов Uniswap V3 с использованием WebSocket

## Описание
Скрипт для отслеживания событий Swap, Mint и Burn в пулах Uniswap V3 в реальном времени. 
Позволяет наблюдать за торговой активностью в выбранном пуле и сохранять данные для дальнейшего анализа.

## Настройка
1. В файле `constants/networks_data.json` в поле `rpc_url` вставьте свой WebSocket URL (начинается с ws:// или wss://)
2. В файле `config/settings.json` настройте:
   - network: Сеть для мониторинга (сейчас поддерживается Ethereum)
   - token1, token2: Тикеры токенов пула (ETH, USDC)
   - proxy: Прокси для подключения (опционально)

## Функции
- Мониторинг событий Swap, Mint и Burn
- Автоматическое определение доступных пулов с разными комиссиями
- Форматирование и отображение данных в удобном виде
- Сохранение событий в CSV файлы для дальнейшего анализа
- Автоматическое переподключение при потере соединения

## Запуск
```
python main.py
```
